String.hasSubString(substr,filter)

判断是否存在子字符串。

参数

substr
必选项。要查找的子字符串。
filter
可选项。字符串型，指定字符串已被 filter 字符分隔。

返回值

布尔型。true/false

描述

该方法是从 String.prototype.indexOf 方法封装而来。

示例

<script type="text/javascript">
var str = "北京2008奥运会";
alert( str.hasSubString("奥运") ); //true
var str2 = "北京,奥运会,中国,北京2008奥运会,奥林匹克";
alert( str2.hasSubString("奥运",",")); //false
alert( str2.hasSubString("北京",",")); //true
</script>
