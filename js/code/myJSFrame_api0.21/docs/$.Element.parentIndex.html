$(Element).parentIndex(Node)

向上查找父节点。

参数

Element
必选项。页面元素ID(字符串型)或者是文档节点(对象)。
Node
必选项。文档节点。

返回值

整型数字。
-1： 表示 Node 节点不在 $(Element) 的所有父节点队列中。
0 ： Node === $(Element)。Node 与 $(Element) 是同一个节点 。
n ： Node 是 $(Element) 向上的第 n 级父节点。

描述

该方法用于在 $(Element) 所有的父节点（一直到文档根节点 document.documentElement）队列中，查找 Node 所在的位置。

示例

<div id="box">
    <div id="box2">
         <div id="box3">demo</div>
    </div>
</div>
<input type="button" value="查找" onclick="alert( $('box3').parentIndex($('box')) )" /> // 2
