Array.uniqStr()

去除字符串型数组里重复的元素。

参数

无

返回值

被去除的重复元素组成的新数组。

描述

该方法仅适用于字符串型数组（数组的所有元素类型均为字符）。

示例

<script type="text/javascript">
var arr = ["one","world","one","dream"];
var repeat = arr.uniqStr();
alert( arr.toString() );   // ["one","world","dream"];
alert( repeat.toString() );  //["one"]
</script>
