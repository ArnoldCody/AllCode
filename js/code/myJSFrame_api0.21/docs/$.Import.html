$.Import(jsFilePath)

 载入 JavaScript 脚本库。

参数

jsFilePath
必选项。脚本库的位置。

返回值

无

描述

该方法相当的后台程序的 include 或者 import 语句，在脚本被解释执行的过程中临时加载并自动预编译一个脚本文件。本方法使用 XMLHTTP 对象并应用其同步方法加载一个 js 文件，然后把脚本字符串作为一个新创建的 script 元素的 text 属性，最后把 script 元素添加到文档的 head 节点中。

需要注意的是，由于使用了 XMLHTTP 对象，因此，目标脚本库必须在同域内才能被加载，并且由于同步的作用，在加载脚本库的时候，页面可能会出现短暂的停顿现象，停顿时间的长短与网速、脚本库的体积有直接关系。

示例

comment.js 文件:
function sayHello(){
    alert("Hello~~~");
}
-------------------- 分割线 --------------------
<script type="text/javascript">
$.Import("comment.js");
sayHello();
</script>
