myJSFrame

myJSFrame 是一款小巧微型的 JavaScript 框架，它有 prototype 的风格，又兼有 JQuery 的味道，它的特点是体积小（目前为15k），使用方便（$(id).show().addClassName("title")）。它在一定程度上改变了传统的 JavaScript 编程风格。它还能识别 CSS 代码（$(id).setStyle("font-size:12px; color:red; margin:1em")）。它增加了 DOM 方法，扩展了 String 与 Array 原型对象，模拟 StringBuffer、NameSpace 类，支持操作 Cookie，浏览器检测，当然，Ajax 还是它的强项，$ 函数支持的 CSS 选择符也不弱 ……

功能扩展：
  • DOM 扩展
  • String 原型扩展
  • Array 原型扩展
  • Function 原型扩展
内置模块：
  • Ajax
  • CSS 选择符
  • StringBuffer 缓冲器
  • NameSpace 模拟命名空间
  • Cookie 操作
  • 表单模拟
  • 浏览器检测
