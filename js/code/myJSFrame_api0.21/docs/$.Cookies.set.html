$.Cookies.set(name,value[,expires[,path[,domain[,secure]]]])

设置指定名称的 Cookie 。

参数

name
必选项。指定 Cookie 名称。
value
必选项。指定 Cookie 的值。
expires
可选项。指定 Cookie 的存活时间。如果为空，存活时间为当前会话结束的时间。类型：大于零的数字，单位：天。
path
可选项。指定 Cookie 生效的路径，子路径能访问父路径的 Cookie 。
domain
可选项。指定 Cookie 生效的站点域。父域的 Cookie 能够被子域访问。
secure
可选项。指定 Cookie 是否被加密传输。值为“secure”或为空。

返回值

null 。

描述

该方法是通过设置 document.cookie 的值实现 set 方法。

示例

<script type="text/javascript" >
     $.Cookies.set("userInfo","acai|26|male",7);  //Cookie 名称为 userInfo，值为 acai|26|male，存活期 7 天。
</script>
