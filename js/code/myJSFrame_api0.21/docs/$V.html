$V(elemORid)

 取控件或元素的值。

参数

elemORid
文档元素或元素的ID。

返回值

文档元素的内部代码或控件的值（innerHTML 或 value）。

描述

该方法只读。用于便捷地取到 $("id").value 或 $("id").innerHTML 。

示例

<div id="abc">info</div>
<input type="text" id="name" value="acai" />
<script type="text/javascript">
alert( $V("abc") ) //info
alert( $V("name") ) //acai
</script>
