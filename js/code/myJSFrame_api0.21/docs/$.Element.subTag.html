$(Element).subTag(tagName)

查找指定子标签的元素。

参数

Element
必选项。页面元素ID(字符串型)或者是文档节点(对象)。
tagName
必选项。元素的标签名。

返回值

所有子标签符合条件的元素数组。

描述

该方法是 getElementsByTagName 的简写方法。

示例

<div id="box">
   <h1>姓名</h1>
   <ul>
       <li>阿蔡</li>
       <li>张三</li>
       <li>李四</li>
   </ul>
</div>
<input type="button" value="获取subTag" onclick="alert( $('box').subTag('li').length )" /> // 3 ( 阿蔡、张三、李四)
