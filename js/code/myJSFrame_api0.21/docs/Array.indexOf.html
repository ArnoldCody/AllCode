Array.indexOf(x)

在数组中查找目标元素。

参数

x
必选项。查找的目标。任意类型。

返回值

-1：数组中不存在目标元素。
0 ：数组的第一个元素等于目标元素。
n ：数组下标为 n 的元素等于目标元素。

描述

该方法通过扩展 Array 的原型对象达到 Array 类型的对象都继承到该方法。Array.prototype.indexOf

示例

<script type="text/javascript">
var arr = ["go","to","hell"];
alert( arr.indexOf("to") );  // 1
</script>
