$(Element).setStyle(cssString)

设置元素的样式。

参数

Element
必选项。页面元素ID(字符串型)或者是文档节点(对象)。
cssString
必选项。样式代码。

返回值

元素的引用

描述

设置元素的样式。样式字符串为普通的 CSS 代码。

示例

<div id="box" style="width:200px"><div class='title'><span>&times;</span>标题</div>content....</div>
<input type="button" value="设置浮动样式" onclick="$('#box span')[0].setStyle('float:right; font-size:11px;')" />
<input type="button" value="设置标题样式" onclick="$('#box div.title')[0].setStyle('font-size:12px; padding:2px; background-color:#ccc')" />
<input type="button" value="设置边框与背景色" onclick="$('box').setStyle('height:300px; border:1px solid #999;background:#eeeeee ')" />
<input type="button" value="设置页面居中" onclick="$('box').setStyle('position:absolute;left:50%; top:50%; margin-left:-100px; margin-top:-150px;')" />
