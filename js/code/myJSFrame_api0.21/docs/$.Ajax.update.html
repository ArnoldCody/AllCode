$.Ajax.update(url,ID)

发送异步请求，并把结果输出到指定(ID)的容器中。

参数

url
必选项。数据发送的目标地址，可以在 url 中携带数据。
ID
必选项。请求成功时结果输出的容器ID。

返回值

当前使用的 xmlhttp 对象。

描述

该方法可以非常方便的更新某一容器的内容。这里的容器可以是 div、span、p、td 等结构元素，也可以是表单元素：input、textarea、select、button，该方法会自动判断 ID 的 tag 类型，并自动选择给 innerHTML 或者 value 赋值。
$.Ajax.update 是 $.Ajax.get 方法的快捷调用方法 。

示例

<div id="content">Loading...</div>
<input id="username" type="text" readonly />
<script type="text/javascript" >
     $.Ajax.update("http://localhost/content.asp?type=getdetail","content");
     $.Ajax.update("http://localhost/user.asp?id=000001","username");
</script>
