$(Element).show()

显示指定的页面元素。

参数

Element
必选项。页面元素ID(字符串型)或者是文档节点(对象)

返回值

元素的引用

描述

设置指定元素的内联样式 display 为空值，并返回元素的引用。注意，该方法只能显示内联样式已设为 display:none 的元素。如果 display:none 存在于外联或外嵌样式表，则该方法的关键语句 display="" 无效，因为 display 等于空相当于什么也没执行。

例子

<div id="box" style="display:none" >内容......</div>
<input type="button" value="显示" onclick="$('box').hide()" />
