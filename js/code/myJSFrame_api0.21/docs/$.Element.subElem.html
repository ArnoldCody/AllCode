$(Element).subElem(CssSelector)

在一元素里用CSS选择符查找子元素。

参数

Element
必选项。页面元素ID(字符串型)或者是文档节点(对象)。
CssSelector
必选项。CSS选择符。

返回值

所有符合CSS选择条件的元素数组。

描述

该方法可在一个未知的DOM节点上，向下用CSS选择符取子元素，比如 $(document.body.firstChild).subElem("input[type=text]");

示例

<script type="text/javascript">
function getTitle(x){
    var title = $(x).subElem("strong.t")[0];
    alert(title);
}
</script>
<div id="title"><strong class="t">I have a dream!</strong>Written by <strong>Martin Luther King, Jr.</strong></div>
<input type="button" value="获取Title" onclick="getTitle('title')" />
