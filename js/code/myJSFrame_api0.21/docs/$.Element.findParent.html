$(Element).findParent(filter)

向上根据条件查找父节点。

参数

Element
必选项。页面元素ID(字符串型)或者是文档节点(对象)。
filter
必选项。过滤条件，其类型为名称值对的对象。

返回值

Node 向上找到的父节点。
null 找不到结果。

描述

该方法用于在 $(Element) 所有的父节点（一直到文档根节点 document.documentElement）队列中，根据条件查找 Node 所在的位置。

示例

<script type="text/javascript">
function getContainer(id){
    var p =$(id).findParent({className:"container", myAttribute:"hello", tagName:"DIV"});
    alert(p?"I find it!":"sorry!");
}
</script>
<div class="container" myAttribute="hello">
    <div class="lefter">
         <div id="news">demo</div>
    </div>
</div>
<input type="button" value="查找" onclick="getContainer('news')" />
