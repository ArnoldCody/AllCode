Array.each(function)

数组的每一个元素都被传入指定的 funtion 执行。

参数

function
必选项。指定的方法。

返回值

原数组。

描述

除了数组的每一个元素都被传入指定的 funtion 执行外，传入 function 的还有一个计数器 i，该计数器是 each 方法内部用于循环数组的计数器。

示例

<script type="text/javascript">
var arr = [0,2,4,6,8,10];
var newArr = [];
arr.each(function(n,i){newArr[i]=++n;})  //数组的每一项都加1，并赋给新数组。
alert( newArr ); // 1,3,5,7,9,11
</script>
