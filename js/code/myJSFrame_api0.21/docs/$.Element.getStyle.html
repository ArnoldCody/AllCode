$(Element).getStyle(styleName)

获取元素的指定样式。

参数

Element
必选项。页面元素ID(字符串型)或者是文档节点(对象)。
styleName
必选项。样式名称。

返回值

元素的引用

描述

获取元素的指定样式。样式名称为css范畴的原始名称。

示例

<div id="box" style="float:right; margin-right:3em " >内容......</div>
<input type="button" value="获取浮动样式" onclick="alert($('box').getStyle('float'))" /> // right
<input type="button" value="获取外补丁样式" onclick="alert($('box').getStyle('margin-right'))" /> // 3em
