$(Element).nextElement()

获取当前节点的下一个同级节点。

参数

Element
必选项。页面元素ID(字符串型)或者是文档节点(对象)。

返回值

下一节点的引用。如果没有下一节点，则为 null 。

描述

该方法是为补充 DOM 的 nextSibling 方法存在浏览器差异而设。

示例

<div id="box">内容......</div>
间隔文字.....
<div id="footer">底部......</div>
<input type="button" value="下一个元素" onclick="alert( $('box').nextElement().id ) " />  // footer
