$.Redirect(URL,QueryString)

 重定向到指定的URL。

参数

URL
必选项。定向的目标位置。
QueryString
可选项。参数名称。

返回值

无

描述

该方法主要是模拟后台程序(asp/asp.net)的 Response.Redirect 方法，获取URL中的参数，它是通过组合 location.href 中的查询参数而实现。

示例

//假如当前页面的URL地址为：http://localhost/product.asp?type=cd (卖CD的列表页面)
<script type="text/javascript">
$.Redirect(location.href,"type=clothes&page=2")  //卖衣服的列表页面，并转到第2页。
</script>
//产品搜索
<input type="text" id="productName" />
<input type="button" value="搜索"
       onclick="$.Redirect(location.href,$.Form.serialize('name':$V('productName')))" />
//跳转的url是:http://localhost/product.asp?type=cd&name=value
