String.hasSubStrInArr(strArr)

判断在字符串型数组中是否存在某元素，该元素被 String 包含。

参数

strArr
必选项。字符串型数组。

返回值

布尔型。true/false

描述

该方法是 hasSubString 的扩展，hasSubString 方法只能传入一个子目标字符串，而 hasSubStrInArr 能传入一组子目标字符串，在这组子目标字符串中只要存在一个元素被 String 包含，就返回 true；如果全部元素都不被 String 包含，返回 false 。

示例

<script type="text/javascript">
var str = "北京2008奥运会";
var strArr = ["广东","北京","上海"];
alert( str.hasSubStrInArr(strArr) ); //true
var strArr2 = ["one world, one dream"];
alert( str.hasSubStrInArr(strArr2) ); //false
</script>
