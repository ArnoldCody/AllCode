String.trim(strORreg)

去除字符串左右两端的空白字符(包括空格、制表符、换页符等)。

参数

strORreg
可选项。默认值是空白字符。当该参数是其他字符时，trim 方法将在字符串两端去除指定的字符strORreg。strORreg 还可以是正则表达式直接量，trim 方法将在字符串里作正则匹配替换。

返回值

两端去除空格后的新字符串。

描述

该方法主要用于去除字符串左右两端的空格，为了扩展，也支持传入参数，用于精确匹配字符串的两端。

示例

<script type="text/javascript">
var title = "          myJSFrame     ";
alert( title.trim() ); //myJSFrame
var author = "===acai=======";
alert( author.trim("=") ); //acai
var build = ".2.0.1.";
alert( build.trim(/(^\.*)|(\.*$)/g) ); //2.0.1
</script>
